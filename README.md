# ✍️ 666 写作灵感生成器 (Writing Prompts Generator)

这是一个基于纯前端技术（HTML/CSS/JS）构建的轻量级网页应用。它从李成豪先生的《666件可写的事：创意无限的写作游戏书》中提取了 666 个极具创意的写作题目，并在每次点击时为你随机抽取一个，帮你打破写作瓶颈，激发创作灵感。

## ✨ 项目特色

* 🎲 **随机抽取**：每次点击即可从 666 个精心设计的题目中获取全新灵感。
* 📋 **一键复制**：内置复制按钮，方便将题目快速粘贴到你的笔记或写作软件中。
* ☁️ **零部署成本**：无后端纯静态页面，完美支持 GitHub Pages 免费托管。
* 📱 **优雅的 UI 设计**：采用类纸张质感的配色与排版，沉浸式阅读，支持手机/电脑端自适应。
